#Бот конветатор ютуба в mp3

Телеграм-бот, который конвертирует видео с ютубчика в аудиофайлы MP3.

Просто отправь ссылку на видео и получи готовый файл


##Функции
- Принимает ссылки вида:
  - `https://www.youtube.com/watch?v=...`
  - `https://youtu.be/...`
- Скачивает.
- Конвертирует в MP3 (128 kbps).
- Присылает готовый файл в чат (до 50 МБ).

###Клонируй репозиторий
```bash
git clone https://github.com/winesses/musicc-bott.git
cd musicc-bott.git
```

###Библиотеки
- pyTelegramBotAPI
- yt-dlp
- requests
- os
- pytube

###Примечания
Бот не обходит ограничения доступа (например, приватные или платные видео).

Размер отправляемого файла не может превышать 50 МБ – ограничение Telegram.

При ошибках бот вежливо объяснит причину.
